solvency2_data.scraping.eiopa_link(ref_date: str, data_type: str = 'rfr') → str[source]#

This returns the link for downloading the selected type of data for a given date Utilises regex to allow for variability in EIOPA publications

Example:

from datetime import date import pandas as pd dr = pd.date_range(date(2016,1,31), date(2023,5,31), freq=’M’) rfr_links = {i.strftime(‘%Y%m%d’): eiopa_link(i.strftime(‘%Y-%m-%d’)) for i in dr} sym_adj_links = {i.strftime(‘%Y%m%d’): eiopa_link(i.strftime(‘%Y-%m-%d’), ‘sym_adj’) for i in dr}

Parameters:

  • ref_date – reference date

  • data_type – type of the dataset
